# Hr Rollout Insights Agent

An AI agent that provides insights on documents related to specific HR rollouts using RAG over SharePoint and multiple document sources

## Overview

Uses A2A Protocol, LangGraph, LiteLLM, and SAP Cloud SDK.

## Structure

- `app/main.py` - A2A server entry
- `app/agent_executor.py` - Request handling
- `app/agent.py` - Agent logic
